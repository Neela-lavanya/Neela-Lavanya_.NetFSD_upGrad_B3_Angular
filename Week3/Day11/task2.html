1. What is jQuery?
jQuery is a fast, small, and lightweight JavaScript library used to simplify HTML DOM manipulation, event handling, and animations.
2. What are the uses of jQuery?
jQuery is used for DOM manipulation, handling events, creating animations, performing AJAX requests, and simplifying JavaScript code.
3. What is an anonymous function?
An anonymous function is a function without a name. It is often used inside events like click() in jQuery.
4. What is DOM traversal?
DOM traversal means navigating between elements in the HTML structure using methods like next(), parent(), children(), etc.
5. What is DOM manipulation?
DOM manipulation means changing the content, structure, or style of HTML elements dynamically using methods like text(), html(), addClass(), removeClass().
